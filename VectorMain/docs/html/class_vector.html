Подробное описание

Класс для работы с векторами.

Класс реализует операции:

  • сложение, вычитание
  • умножение вектора на число и векторное произведение
  • вычисление длины
  • вычисление косинуса между векторами
  • сравнение длин векторов
  • ввод и вывод через потоки
